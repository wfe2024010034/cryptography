 1. 实验目的
使用穷举法破解凯撒密码密文，掌握凯撒密码的加解密原理，熟悉穷举法的编程实现。

 2. 核心结论
- 正确密钥 $k$：9
- 解密后的明文：ILOVEPROGRAMMINGVERYMUCH

 3. 判断依据
凯撒密码解密的核心是通过偏移量还原有意义的英文文本：
1. 穷举1~25所有密钥后，仅当 $k=9$ 时，解密结果是符合英文语义的完整句子（ILOVE PROGRAMMING VERY MUCH，即“我非常喜欢编程”）；
2. 其他密钥的解密结果均为无意义的字母组合（如k=1的MTEDBLAVXTILAHOEXMAXVHWX），无法构成合理语义；
3. 明文由常见英文单词/短语组成，符合凯撒密码解密后“有意义文本”的核心判断标准。

 4. 解密原理
凯撒密码是替换加密的一种，解密时将每个字母按字母表反向偏移 $k$ 位：
- 公式：$明文字符 = (密文字符ASCII码 - 'A'的ASCII码 - k) \% 26 + 'A'的ASCII码$
- 取模运算保证偏移后仍在A-Z范围内（避免超出字母表边界）。
